# Support

- For repository bugs or documentation errors, use the repository's issue tracker.
- For benchmark or pricing updates, include reproducible data and a dated public source.
- For APIMART API usage, consult [the documentation](https://docs.apimart.ai/).
- For account, billing, quota, or service incidents, use the support channel on the official APIMART website.

Never post live API keys, customer data, or private request logs in a public issue.
